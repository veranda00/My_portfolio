# Анализ данных для агентства недвижимости

Агентству нужен готовый анализ объявлений о продаже жилой недвижимости в Санкт-Петербурге и Ленинградской области, чтобы найти самые перспективные сегменты недвижимости.

После знакомства с данными были решены следующие ad hoc задачи с помощью SQL-запросов:  
* Определены наиболее привлекательные сегменты недвижимости Санкт-Петербурга и городов Ленинградской области, чтобы создать эффективную бизнес-стратегию на рынке недвижимости.
* Поняли сезонные тенденции на рынке Санкт-Петербурга и городов Ленинградской области, чтобы знать периоды с повышенной активностью продавцов и покупателей недвижимости в регионе. 
* Выяснили, в каких населённых пунктах Ленинградской области активнее всего продаётся недвижимость и какая именно, чтобы знать перспективные районы.
